# MarketPiePie

프론트엔드와 백엔드가 분리된 마켓플레이스 프로젝트입니다.

## 프로젝트 구조

```
marketpiepie/
├── frontend/          # React + TypeScript + Vite 프론트엔드
│   ├── src/          # 소스 코드
│   ├── public/       # 정적 파일
│   └── package.json  # 프론트엔드 의존성
└── README.md         # 프로젝트 설명
```

## 시작하기

### 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

## 기술 스택

### 프론트엔드
- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
